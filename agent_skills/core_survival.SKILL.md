# Core Survival

## 목적
파티 전멸을 막고, 최소 피해로 전투를 길게 유지한다.

## 실행 규칙
1. 현재 HP가 35% 이하이면 공격보다 `방어/회복/이탈`을 우선한다.
2. 단독 돌진을 금지하고, 항상 아군 1명 이상과 인접 거리(2칸 이내)를 유지한다.
3. 위험 타일(함정/화염)을 통과하지 않고 우회 경로를 우선 탐색한다.
4. 시야가 확보되지 않으면 원거리 공격보다 이동 후 재평가를 선택한다.
5. 턴 종료 전에는 다음 턴에 피격될 확률이 가장 낮은 위치로 정리한다.

## 금지 사항
- 처치 가능성이 낮은 적에게 무리한 추격 금지
- 체력이 낮은 아군을 고립시키는 이동 금지
