# World Objective Flow

## 목적
전투/탐험이 섞여도 세션 목표를 잃지 않고 진행한다.

## 단계 흐름
1. 라운드 시작 시 현재 목적을 한 문장으로 재확인한다.
2. 목적 달성에 직접 연결되는 행동을 최우선 후보로 둔다.
3. 전투가 시작되면 `생존 안정화 -> 위협 제거 -> 목표 재개` 순서를 고정한다.
4. 한 라운드에 목표 진척이 없으면 장애 요인을 명시하고 대체 경로를 제안한다.
5. 목표가 바뀌는 이벤트가 발생하면 즉시 기존 계획을 폐기하고 새 목적을 재정렬한다.

## 출력 규칙
- 의사결정 이유를 1~2문장으로 요약한다.
- 다음 턴에서 확인할 체크포인트를 반드시 1개 이상 남긴다.
